# 📚 Dépôt des Projets et Plans de Soutenance

Bienvenue sur ce dépôt GitHub.  
Ce repository permet de centraliser :

- 📘 Les plans de livres de soutenance
- 💻 Les projets des étudiants
- 📄 Les documents liés aux projets

Merci de respecter l’organisation demandée afin de garder un dépôt propre et facile à consulter.

---

# 📁 Structure Obligatoire

Dans la branche principale (`main`), chaque étudiant doit créer **un dossier personnel** sous le format :

```text
Nom_Matricule
```

## ✅ Exemples

```text
Rakoto_2456
Rabe_1987
Andria_3021
```

---

# 📂 Contenu du dossier personnel

Chaque dossier doit contenir :

- Le plan de soutenance
- Le projet
- Les documents nécessaires

## ✅ Exemple de structure

```text
Rakoto_2456/
│
├── Plan_Soutenance.pdf
├── Projet/
│   ├── source_code/
│   ├── documentation/
│   └── assets/
└── README.md
```

---

# ⚠️ Règles Importantes

- ❌ Ne modifiez pas les dossiers des autres étudiants
- ✅ Respectez le format :

```text
Nom_Matricule
```

- ❌ Évitez les espaces dans les noms de dossiers
- ✅ Vérifiez vos fichiers avant le push
- ✅ Organisez correctement vos documents

---

# 🚀 Comment Ajouter Votre Dossier

## 1️⃣ Cloner le repository

```bash
git clone URL_DU_REPOSITORY
```

---

## 2️⃣ Créer votre dossier

```bash
mkdir Nom_Matricule
```

---

## 3️⃣ Ajouter vos fichiers

Placez votre plan de soutenance et votre projet dans votre dossier personnel.

---

## 4️⃣ Envoyer les modifications

```bash
git add .
git commit -m "Ajout du dossier Nom_Matricule"
git push
```
